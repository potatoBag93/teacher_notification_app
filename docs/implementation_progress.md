**파일**: `src/components/NoticePreview.vue` (653줄)

#### 핵심 기능
- ✅ **요소 표시/숨김 제어**
  - 헤더 영역 표시 여부
  - 항목 번호 표시 여부  
  - 개별 항목 내용 표시 여부
  - 실시간 토글 기능

- ✅ **새창 미리보기**
  - 편집 화면과 독립된 미리보기 창
  - 줌 컨트롤 (50% ~ 300%)
  - 키보드 단축키 지원 (Ctrl+/-/0)
  - 프린트 최적화 스타일

- ✅ **반응형 디자인**
  - 모바일/태블릿/데스크톱 최적화
  - 터치 인터페이스 지원
  - 다양한 화면 크기 대응
